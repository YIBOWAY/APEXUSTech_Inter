## 摘要

本报告呈现了对 V4.0 版本量化交易策略的研究成果。V4 策略是在 V3 动态现金管理模型基础上的重大迭代。相较于 V3 版本的保守属性，V4 通过优化 VIX 阈值、降低动量触发门槛、引入信念加权 (Conviction Weighting) 以及有条件空头对冲 (Conditional Short Overlay)，实现了收益潜力的深度挖掘。

研究结果显示，V4 策略在 68 个月的全样本回测中，年化收益率提升至 36.05% (V3 为 23.37%)，Sharpe 比率为 1.306。虽然策略在攻击性提升的同时伴随了回撤的上升 (MaxDD -22.29%)，但其风险收益比依然显著优于 XLK 基准 (Sharpe 1.017, MaxDD -31.21%)。本研究详细探讨了五大核心改动对策略表现的影响，并验证了其在不同市场环境下的鲁棒性。

## 1. 项目目标与技术栈

### 1.1 研究目标
1. **收益增强**：通过放宽持仓限制和优化现金分配公式，捕获科技股在低波动环境下的超额收益。
2. **权重精细化**：摒弃 V3 的等权重分配，实施基于动量强度 (z-score) 的信念加权模型。
3. **防御机制演进**：引入条件空头，在下行趋势且波动率受控的特殊环境下，通过做空弱势股对冲风险。
4. **参数优化**：通过网格搜索确定最佳换手频率 (Holding Period)，平衡交易成本与动量衰减。

## 2. 方法论：V4 的五大核心改动

V4 策略是对 V3 逻辑的彻底重构。其核心思想是：在确定性强的环境中加大杠杆，在极端风险中迅速离场，在弱势震荡中实施精准对冲。

### 2.1 优化后的 VIX 波动率门控 (VIX Gating)
V3 在 VIX 处于 20-25 区间时会主动减仓至 75%。V4 研究发现，该区间往往是趋势确认后的主升浪阶段，过度保守导致了收益流失。
- **V4 规则**：在 VIX < 25 时维持 100% 的暴露权重。
- **阶梯阈值**：
  - `<20 & <25`: 权重 1.0
  - `<30`: 权重 0.85
  - `<35`: 权重 0.60
  - `<45`: 权重 0.30
  - `>45`: 权重 0.0 (完全空仓)

### 2.2 降低动量触发阈值
V3 要求动量 z-score 必须大于 1.5 才能获得全额分配，这导致策略在震荡行情中仓位过低。
- **V4 规则**：将全额权重阈值下调至 z > 0.8。
- **灵敏度调节**：
  - `>0.8`: 1.0
  - `>0.4`: 0.90
  - `>0.0`: 0.75
  - `>-0.3`: 0.50
  - `其他`: 0.20

### 2.3 重新定义的仓位计算公式
为了平衡动量、趋势与基础暴露，V4 调整了权益分配系数。
- **V3 公式**：`0.40*w_mom + 0.40*w_trend + 0.20*1.0`
- **V4 公式**：`equity_weight = w_vix * (0.40*w_mom + 0.35*w_trend + 0.25*1.0)`
- **改进逻辑**：增加了 5% 的基础持仓 (Base weight)，同时略微降低对长期趋势 (Trend) 的依赖，使策略对短期价格变动更敏感。

### 2.4 信念加权持仓分配 (Conviction-Weighted)
这是 V4 最显著的数学改进。策略不再对选中的标的进行平摊，而是根据其动量 z-score 的相对强度分配资金。
- **实现逻辑**：
  1. 获取选中个股的原始 z-score。
  2. 对分数进行位移处理 (`score - min + 0.01`) 确保权重为正。
  3. 归一化后乘以总权益权重 (equity_weight)。
- **个人理解**：这种做法体现了“强者恒强”的逻辑。当某只个股展现出极端动量时，策略会给予其高于平均水平的配置，从而在主升浪中获得更高贝塔。

### 2.5 有条件空头叠加 (Conditional Short Leg)
V4 引入了对冲模块。仅在以下特定环境触发：
- **触发条件**：市场处于下行趋势 (Downtrend) 且 VIX < 35。
- **执行方式**：利用闲置现金的 15% 做空动量排名倒数 10% 的股票。
- **个人理解**：当市场阴跌且未发生恐慌踩踏时，做空最弱势的科技股能有效抵消多头头寸的损失。一旦 VIX 飙升至 35 以上，策略会切断该模块以防扎空。

- **低波动环境 (Low Vol)**：运行 10 个月，年化收益率达 **122.21%**，Sharpe 4.449。t 检验结果 (t=2.75, p=0.0226) 表明该环境下的盈利具有极高的统计显著性。

### 5.3 现金分配逻辑验证

这证明了 V4 的 VIX 门控在保持攻击性的同时也保留了必要的审慎。

## 6. 风险评估

1. **波动率上移风险**：V4 的年化波动率达到 25.23%，对投资者的心理承受能力有更高要求。
2. **扎空风险**：有条件空头虽然能对冲阴跌，但在市场发生“V型”反转时，空头头寸可能面临快速亏损。目前的 15% 现金权重是一个折中方案。
3. **自相关性预警**：策略收益率的 Lag 2 自相关性为 -0.2141，表明收益存在一定的短期均值回归特性，连涨后需警惕波动。
4. **流动性风险**：虽然针对 XLK 成份股（如 PLTR, KLAC, STX），但在极端市场环境下，信念加权可能导致部分头寸过大，需配合交易量加权法 (Volume Weighted) 缓解压力。

## 7. 实施建议

1. **首选方案**：推荐采用 **成交量加权法 (Volume Weighted)** 配合 **2 个月持仓周期**。该组合在 Sharpe 比率与回撤控制之间取得了最佳平衡。
2. **动态调仓**：应根据 V4 提出的 VIX 阶梯门控进行严格的仓位管理，切勿在 VIX > 45 时强行持仓。
3. **多空平衡**：空头模块的标的选股应严格限制在动量倒数 10% 且成交量活跃的个股中，避免因流动性匮乏导致的撤单困难。

# V4 动态权重分配逻辑
def calculate_position_size_v4(vix_level, momentum_signal, trend_strength):
    # VIX 门控 (更激进)
    if vix_level < 25: w_vix = 1.0
    elif vix_level < 30: w_vix = 0.85
    elif vix_level < 35: w_vix = 0.60
    elif vix_level < 45: w_vix = 0.30
    else: w_vix = 0.0

    # 动量信号 (降低门槛)
    if momentum_signal > 0.8: w_mom = 1.0
    elif momentum_signal > 0.4: w_mom = 0.90
    elif momentum_signal > 0.0: w_mom = 0.75
    else: w_mom = 0.50

    # V4 综合权益比例
