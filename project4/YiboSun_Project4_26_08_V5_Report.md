## 摘要

本报告呈现了 V5.0 版本量化交易策略的研究成果。V5 策略是对 V4.0 “信念加权”逻辑的深度进化，其核心创新在于引入了“双因子 VIX 体制分类器 (Dual-factor VIX Regime Classifier)”。该机制通过结合 VIX 的时间占比 (Time-Proportion) 与 VIX/VIX3M 的期限结构 (Term Structure)，克服了 V4 单点阈值触发易受短期噪音干扰的缺陷。

研究结果显示，V5 策略在 68 个月的全样本回测中，年化收益率提升至 39.36% (V4 为 36.05%)，Sharpe 比率显著优化至 1.544 (V4 为 1.306)。即便在 V4 的高收益基础上，V5 依然通过更精准的风险感知实现了约 2.4 个百分点的年化超额收益提升，同时维持了回撤控制能力 (MaxDD -23.62%)。本研究验证了体制分类模型在动态资产配置中的卓越价值。

## 1. 项目目标与技术栈

### 1.1 研究目标
1. **波动率感知增强**：从 V4 的单点阈值切换到基于“时间比例”与“期限结构”的双因子体制分类模型。

3. **前瞻性防御**：利用 VIX/VIX3M 期限结构捕捉市场从正向市场 (Contango) 向反向市场 (Backwardation) 的转变，提前识别系统性风险。

## 2. 方法论：V5 的双因子体制分类

V5 策略在保留 V4 “信念加权”、“条件空头”及“低动量阈值”的基础上，对现金管理模块进行了底层重构。

### 2.1 双因子体制分类逻辑 (Dual-Factor Regime)
V5 认为，单一的 VIX 数值无法区分“短暂的情绪闪崩”与“系统性的持续动荡”。

*   **因子 1 - 时间占比 (Time-Proportion)**：
    *   动态阈值：`thr_vix = max(20, rolling_252d_quantile(VIX, 0.75))`。
    *   若当月中超过 25% 的交易日 VIX 高于该阈值，则触发 `high_time` 信号。这能有效识别波动率中枢的实质性上移。
*   **因子 2 - 期限结构 (VIX/VIX3M Ratio)**：
    *   通过 VIX 与 VIX3M 的比率识别市场结构。
    *   `flat_elev` (r_avg > 0.97)：结构趋于平坦。
    *   `inv_panic` (r_avg > 1.00)：进入贴水状态 (Backwardation)，预示真实恐慌。

### 2.2 三大体制定义与权益配置
| 体制类型 (Regime) | 触发条件 | 现金管理系数 (w_vix) |
| :--- | :--- | :---: |
| **Normal (常态)** | 两个因子均未触发 | 1.00 |
| **Elevated (中等风险)** | 仅一个因子触发 (XOR) | 0.75 |
| **Panic (恐慌)** | 两个因子同时触发 (AND) | 0.35 |

**回退机制**：若 VIX3M 数据缺失，策略将自动将权重上限限制在 Elevated 等级，确保安全。

### 2.3 核心计算公式 (沿用 V4 框架)
`equity_weight = w_vix * (0.40 * w_mom + 0.35 * w_trend + 0.25 * 1.0)`

## 6. 风险评估

1. **体制切换滞后**：由于分类器基于月度频率，在极端“闪崩”发生的当月，策略可能无法即时切换至 Panic 体制，存在净值短期波动的风险。
2. **VIX3M 依赖**：策略高度依赖 VIX3M 数据。若该数据源失效，回退至 Elevated 的保护机制虽安全但会牺牲部分极端环境下的收益。
3. **回撤控制上限**：尽管引入了更精细的体制分类，最大回撤仍维持在 -23% 左右，表明动量策略在科技板块高 Beta 属性下的天然波动无法完全消除。

### A. 双因子体制分类伪代码
```python
def compute_monthly_vix_regime_v5(vix_series, vix3m_series):
    # 因子 1: 时间占比
    thr_vix = max(20, vix_series.rolling(252).quantile(0.75))
    high_time = (vix_series > thr_vix).mean() > 0.25
    
    # 因子 2: 期限结构
    ratio = vix_series / vix3m_series
    flat_elev = ratio.mean() > 0.97
    inv_panic = ratio.mean() > 1.00
    
    # 体制逻辑
    if high_time and inv_panic:
        return "Panic", 0.35
    elif high_time or flat_elev:
        return "Elevated", 0.75
    return "Normal", 1.00
```
